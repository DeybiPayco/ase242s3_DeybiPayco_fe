# ❓ ¿Qué es Angular?

Angular es un framework de desarrollo web de código abierto desarrollado y mantenido por Google.  
Se utiliza para crear aplicaciones web dinámicas, modernas y de una sola página (SPA - Single Page Applications).

Angular está basado en:

- TypeScript
- Arquitectura basada en componentes
- Enlace de datos bidireccional (Two-way data binding)
- Inyección de dependencias

Permite desarrollar aplicaciones escalables, mantenibles y estructuradas.

---

# 🛠️ ¿Qué herramientas necesitan instalar?

Para trabajar con Angular es necesario instalar las siguientes herramientas:

## 1️⃣ Node.js

Es el entorno de ejecución que permite usar npm (Node Package Manager).

Descargar desde:
https://nodejs.org/

Verificar instalación:

```bash
node -v
npm -v
```

---

## 2️⃣ Angular CLI

Es la herramienta oficial para crear y administrar proyectos Angular.

Instalación global:

```bash
npm install -g @angular/cli
```

Verificar instalación:

```bash
ng version
```

---

## 3️⃣ Editor de código

Se recomienda:

- Visual Studio Code

Descargar desde:
https://code.visualstudio.com/

---

## 4️⃣ Crear un proyecto Angular

```bash
ng new nombre-proyecto
cd nombre-proyecto
ng serve
```

Abrir en el navegador:

```
http://localhost:4200/
```
